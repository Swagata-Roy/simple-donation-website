DOM Questions & Answers

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents web pages as a tree structure. It allows JavaScript to:

  • Access and change page elements
  • Modify content and styles
  • React to user interactions
  • Create dynamic web experiences
How do you select an element from the DOM?

Common methods to select DOM elements:

  • getElementById('id') - finds by ID
  • getElementsByClassName('class') - finds by class
  • querySelector('.class') - uses CSS selectors
  • querySelectorAll('div') - finds all matching elements
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you handle events on a parent element instead of individual child elements. Benefits:

  • Better memory usage
  • Works with dynamic elements
  • Less code needed
  • Easier maintenance
How do you manipulate element attributes and styles using the DOM?

Elements can be manipulated in several ways using the DOM:

  • element.setAttribute('class', 'new-class')
  • element.style.color = 'red'
  • element.classList.add('active')
  • element.textContent = 'New text'
